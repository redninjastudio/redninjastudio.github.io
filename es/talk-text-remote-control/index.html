Talk Text (Control Remoto)


Esta es una aplicación de control remoto para Talk Text (Texto A Voz) que puedes usar online (.jnlp) en cualquier ordenador con java 8+, en Windows (.exe y .msi), o en GNU/Linux (.deb)

¿Por qué la versión .jnlp usa all-permits en vez de sandbox? Simplemente porque el comportamiento actual de sandbox impide aleatóriamente la conexión al mismo servidor y puerto de donde se descarga la aplicación en GNU/Linux, puede que por no ser una IP fija o puede que por un bug.
